# reactive-icap-client

A Reactive ICAP Client for Java.

## Status

This project is in an early planning stage. No implementation exists yet — only
the repository scaffolding (license, CI/security configuration) is in place so
far. Check back later, or open an issue if you're interested in contributing
or have requirements you'd like to see covered.
